API Documentation

The following endpoints allow interaction with the resource:

  • POST /api/resource: Creates a new resource. Returns a fixed ID of 12345 with a 201 status code.
  • GET /api/resource/:id: Retrieves a resource by its ID.
    • Returns a 200 and the resource data if :id is 12345.
    • Returns a 404 if the resource with the specified ID does not exist.
